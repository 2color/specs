The `prisma init` command is only permitted in **empty directories**. In the future, we can try to be smart about detecting certain files and inferring information about the user's environment, but for now we're going with a simple approach to ensure people don't hit roadblocks with existing files.

If `prisma init` is executed in a non-empty directory and no project directory name is provided as an argument, the command will fail immediately:
